## Account Security First

### Seed Phrase Safety
Your seed phrase is your **key to everything**. Lose it = lose your NFTs and DOT forever.

**DO:**
- ✅ Write it down on paper (multiple copies)
- ✅ Store in a secure location (safe, bank vault)
- ✅ Never share with anyone
- ✅ Test restore process with small amount first

**DON'T:**
- ❌ Screenshot or save digitally
- ❌ Share via messages/email
- ❌ Store in cloud services
- ❌ Tell anyone your seed phrase

### Account Types

**Sr25519 (Recommended)**
- Default Polkadot account type
- Best for most users
- Compatible with all parachains

**Ed25519**
- Alternative signature scheme
- Some advanced use cases
- Full compatibility

## Backup & Recovery

### Export Account
1. Open wallet extension
2. Click three dots next to account
3. Select "Export Account"
4. **Save the JSON file securely**
5. Remember your password!

### Import Account
1. Click "Import Account" 
2. Choose method:
   - Seed phrase (most common)
   - JSON file + password
   - QR code

### Test Recovery
Always test with a small amount first:
1. Create new account with test funds
2. Export and delete it
3. Import it back
4. Verify funds are accessible

## Security Best Practices

### Browser Security
- ✅ Use updated browsers only
- ✅ Never use public/shared computers
- ✅ Clear browser data on shared devices
- ✅ Use hardware wallet for large amounts

### Transaction Signing
**Always verify before signing:**
- Recipient address is correct
- Amount is what you expect
- Network fees seem reasonable
- You initiated this transaction

### Phishing Protection
**Red flags:**
- Unexpected transaction requests
- URLs that look like chaotic.xyz but aren't
- Requests to "validate" your wallet
- Urgent messages about account issues
