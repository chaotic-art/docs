## Wallet Setup

### Supported Wallets
- **Polkadot{.js} Extension** (Recommended)
- **Talisman Wallet** (Full-featured)
- **SubWallet** (Mobile + Desktop)

### Install Polkadot{.js} Extension

1. Visit [polkadot.js.org/extension](https://polkadot.js.org/extension/)
2. Download for your browser (Chrome, Firefox, Edge)
3. Create or import your account
4. **Important**: Keep your seed phrase safe!

### Connect to Chaotic

1. Visit [chaotic.xyz](https://chaotic.xyz)
2. Click "Connect Wallet" in the top right
3. Select your wallet from the list
4. Approve the connection request

## Get Some DOT

### Where to Buy DOT
- **Centralized Exchanges**: Binance, Coinbase, Kraken
- **DEX**: Via Asset Hub or other parachains
- **P2P**: LocalCryptos or similar platforms

### Minimum Amounts
- **Trading**: ~2 DOT minimum (includes fees)
- **Minting**: ~5 DOT recommended for gas + deposits
- **Shopping Cart**: ~10+ DOT for bulk purchases

## Troubleshooting

### Common Issues

**"Wallet not detected"**
- Refresh the page
- Make sure extension is unlocked
- Try a different browser

**"Insufficient balance"**
- Check you have enough DOT for transaction + fees
- Use our auto-teleport feature to move funds

**"Transaction failed"**
- Network congestion - try again in a few minutes
- Check if you have existential deposit requirements
